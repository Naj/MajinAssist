Configurer mon agenda
Option recommandée — Lien ICS (vrais événements, pas d'iframe) :
Google : calendar.google.com → ⚙️ Paramètres → votre calendrier → "Intégrer le calendrier" → copiez l'"Adresse secrète au format iCal"
Outlook : outlook.com → Calendrier → Partager → "Publier un calendrier" → copiez le lien ICS
iCloud : icloud.com/calendar → ⋯ sur le calendrier → "Calendrier public" → copiez le lien (commence par webcal://)
Collez ensuite ce lien dans Paramètres → Agenda → "URL ICS" → Valider, puis choisissez "Agenda ICS" dans le mode d'affichage.
Alternative — Iframe Google Agenda :
1. calendar.google.com ↗ → ⚙️ Paramètres → votre calendrier
2. Section "Intégrer le calendrier" → copiez l'URL src de l'iframe
3. Collez dans Paramètres → Agenda → URL iframe → Valider
4. Sélectionnez "Iframe" dans le mode d'affichage

Lancer des apps de bureau depuis le navigateur
Pourquoi c'est nécessaire :
Un navigateur ne peut pas lancer un programme local pour des raisons de sécurité — c'est volontaire. La version Electron de MAJIN contourne ça avec un pont natif intégré. La version web (Cloudflare) ne peut pas en faire autant... sauf si Windows sait rediriger un lien spécial vers un programme local. Chaque appli que vous déclarez ci-dessus obtient son propre protocole dédié (ex. majin-xyz123:), avec une commande figée à l'installation — exactement comme les liens spotify: ou zoommtg: qui n'ouvrent jamais que leur propre appli. La page web ne transmet aucune donnée au moment du lancement, le protocole ne fait qu'exécuter la commande déjà enregistrée.
Installation :
1. Ajoutez chaque appli voulue avec "Ajouter une appli" (nom + commande complète, ex. le chemin du .exe)
2. Cliquez "Télécharger l'installateur (.bat)" — il contient un protocole par appli déclarée
3. Double-cliquez le fichier téléchargé (Windows peut afficher un avertissement SmartScreen pour un .bat inconnu — "Informations complémentaires" puis "Exécuter quand même")
4. Cochez "Installateur exécuté sur ce PC" dans Paramètres
5. Au premier lancement de chaque appli, votre navigateur demandera une confirmation ponctuelle — normal, cochez "toujours autoriser" pour ne plus la revoir
6. Si vous ajoutez ou modifiez une appli plus tard, retéléchargez et relancez l'installateur pour mettre à jour les protocoles
✅ Plus sûr que l'ancienne version :
Aucun PowerShell n'est impliqué, aucune donnée n'est décodée à l'exécution — chaque protocole exécute une commande unique et fixe, choisie par vous au moment de l'installation. La page web ne peut déclencher que les applis que vous avez explicitement déclarées, rien d'autre. Vous pouvez retirer un protocole à tout moment avec "Tout désinstaller".

Ajouter un flux RSS personnalisé
Un flux RSS est une URL spéciale que les sites d'actualité publient pour diffuser leurs articles automatiquement. MAJIN l'utilise pour afficher les titres dans le bandeau.
①Trouver l'URL RSS d'un site
Sur la plupart des sites, cherche un lien RSS ou Flux en bas de page.
Ou ajoute /rss ou /feed à la fin de l'URL du site.
Exemple : https://monsite.fr/rss ou https://monsite.fr/feed.xml
②Sélectionner "URL personnalisée"
Dans le menu Source prédéfinie, choisis ✏️ URL personnalisée…
Le champ URL flux RSS s'active — colle ton URL dedans.
③Actualiser et vérifier
Clique sur Actualiser à droite du champ.
Le statut indique le nombre d'articles chargés.
Si 0 article : l'URL est peut-être incorrecte ou le site bloque les flux.
💡 Astuce : Pour trouver l'URL RSS de n'importe quel site, tape dans Google :
site:monsite.fr RSS feed
